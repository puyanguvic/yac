# YAC Simulation Outputs

This folder stores generated experiment outputs (CSVs and figures). Source code lives in `src/yac_sim/`.

## Run

```bash
PYTHONPATH=src python -m yac_sim
```

Outputs will be saved to `result/`:

- `exp_quant_tradeoff.csv`
- `exp_markov_robustness.csv`
- `exp_two_uav_baselines.csv`
- `fig_quant_tradeoff.png`
- `fig_markov_robustness.png`
- `fig_two_uav_trajectory.png`
